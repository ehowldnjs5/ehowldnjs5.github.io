HTML이란?

HTML은 하이퍼텍스트 마크업 언어 (Hypertext Markup Language)의 약자로, 웹 페이지를 만들 때 사용되는 문서 형식입니다. 하이퍼텍스트(Hypertext)하이퍼링크(Hyperlink)를 담고 있는 문서를 뜻합니다. 하이퍼링크는 문서들이 서로 연결되어 있는 고리를 말합니다.

이미지 넣기

이미지를 넣으려면 img 요소를 사용합니다.

src속성에는 이미지의 경로(URL)를 적습니다.

이미지요소에는 항상 alt 속성을 사용하여 대체 텍스트를 명시해주어야 합니다. "alt"는 "Alternative text"의 약자로, 이미지를 표시 할 수 없을 때 이미지를 대신하여 나타낼 텍스트를 뜻합니다.

CSS란?

CSS는 Casxading Stylesheet의 약자이며 HTML 같은 마크업 언어로 작성된 문서를 꾸미기 위해 사용합니다. HTML 문서의 head 요소안에 style 요소를 넣고 css코드를 적으면 해당 문서를 다양하게 꾸밀 수 있습니다.

CSS에 대한 더 자세한 설명은 CSS문서를 참고하세요.
